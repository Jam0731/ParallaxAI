# Changelog

All notable changes to ParallaxAI will be documented in this file.

## [0.1.0] - 2026-06-16

### Added
- Multi-agent collaboration gateway with @mention routing
- 4 specialized agents: Munger, Woz, Ogilvy, Taleb
- WebSocket real-time communication
- Claude Code and MiMo Code adapter support
- Session persistence (survives page refreshes)
- Multi-workspace support
- Cost tracking with real-time token display
- Knowledge base with FTS5 full-text search
- Cron job scheduler for automated tasks
- Auto-Dream memory consolidation
- Web UI with dark theme: chat, agent status, role management, skill management, task boards
- Slash commands: /compact, /clear, /search, /cost, /status, /export, /help
- @mention autocomplete with keyboard navigation
- Delegation task tracking board
- Scheduled task management board
- SkillHub integration for skill installation
- Agent persona system (AGENTS.md + SKILL.md separation)
- Cross-agent context sharing
- Shared memory system (shared_context.json)
- Error recovery with retry + circuit breaker
- Backup/restore functionality
